================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "git: command not found"
SOLUTION: Install Git from https://git-scm.com/downloads

ISSUE: Authentication failed
SOLUTION: Use GitHub Personal Access Token instead of password
  1. Go to GitHub Settings > Developer settings > Personal access tokens
  2. Generate new token with 'repo' scope
  3. Use token as password when pushing

ISSUE: Remote already exists
SOLUTION: Remove and re-add remote
  git remote remove origin
  git remote add origin https://github.com/yourusername/tejas-raksha.git

ISSUE: Files not showing on GitHub
SOLUTION: Check .gitignore file - some folders are excluded:
  - .venv/
  - __pycache__/
  - scan-results-*/
  - test-scan/
  This is intentional to keep the repository clean.
