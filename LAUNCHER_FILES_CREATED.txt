WHAT EACH LAUNCHER DOES
================================================================================

STEP 1: Activate Virtual Environment
   - Automatically runs venv activation
   - No manual activation needed
   - Checks if venv exists

STEP 2: Get Target URL
   - Prompts: "Target URL: "
   - Validates input
   - Shows examples

STEP 3: Get Scan Depth
   - Prompts: "Enter choice (1/2/3)"
   - Options: Quick/Normal/Deep
   - Default: 1 (Quick)

STEP 4: Get Report Format
   - Prompts: "Enter choice (1/2/3)"
   - Options: HTML/JSON/CSV
   - Default: 1 (HTML)

STEP 5: Confirm and Run
   - Shows configuration summary
   - Asks for confirmation
   - Runs scan with all parameters
   - Opens report automatically
